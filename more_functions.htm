Comparison functions

You can use the function (=) to test the equality of two things 


(defn meaning-of-life?
  [x]
  (= x 42))

Other Comparison functions


(> 4 3)    ;=> true
(>= 4 5)   ;=> false
(< -1 1)   ;=> true
(<= -1 -2) ;=> false
(< 1 5 9)  ;=> true
(< 1 5 3)  ;=> false

String functions

A large part of programming is manipulating strings. (str) makes a list of strings into one string 


(str "Chocolate" ", " "strawberry" ", and " "vanilla")
;=> "Chocolate, strawberry, and vanilla"

Anonymous functions

Functions don't have to have names. This type we call anonymous functions 


(fn [string1 string2] (str string1 " " string2))

Anonymous functions


(map (fn [x] (* 3 x)) [1 2 3]) ;=> [3 6 9]

(reduce (fn [x y] (+ x y)) [1 2 3]) ;=> 6

(reduce
  (fn [s1 s2] (str s1 " " s2))
    ["i" "like" "peanut" "butter" "and" "jelly"])
    ;=> "i like peanut butter and jelly"
